# Pdf Analyzer

PDF document security analyzer.

Penganalisis keamanan dokumen PDF.


## Description

Examines PDF files for embedded scripts, suspicious objects, and potential exploits.

## Deskripsi

Memeriksa file PDF untuk script tertanam, objek mencurigakan, dan potensi eksploit.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Analyze: python pdf_analyzer.py document.pdf
- Demo: python pdf_analyzer.py --demo

## Penggunaan

- Analisis: python pdf_analyzer.py document.pdf
- Demo: python pdf_analyzer.py --demo
